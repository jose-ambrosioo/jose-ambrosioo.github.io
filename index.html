BuyHere E-Commerce Web App

Capstone Project | FHSU Department of Computer Science | Masters in Computer Science

JavaScript | React.js | Express.js | Node.js | MongoDB | JWT | RESTFUL APIs | PayPal REST API | Lucide | ShadCN UI

BuyHere is a comprehensive e-commerce platform built to deliver a seamless and secure online shopping experience. It leverages advanced frontend and backend technologies to ensure responsive design, intuitive navigation, and efficient order management for both customers and administrators.

Sleepify Hotel/Property Booking Web App

Django | Python | JavaScript | HTML | CSS | SQL | RESTful APIs | PayPal REST API

Sleepify is a hotel/property booking platform that offers a seamless and efficient experience for both customers and hotel administrators. Customers can easily browse and book properties, manage their profiles, and leave reviews and ratings. Hotel administrators and property owners are equipped with powerful tools to manage listings, rooms, and bookings effectively.

The platform supports secure user authentication, Paypal booking payments, and hotel reviews and ratings, making the booking process more convenient and trustworthy. Additionally, Sleepify provides user-friendly dashboards for both customers and administrators to easily manage their accounts and operations.

Built with Django for the backend and HTML, CSS, and JavaScript for the frontend, Sleepify ensures a responsive and reliable experience. A SQL-based database is used for efficient data management, providing smooth performance and scalability.

Calories Burned Prediction
